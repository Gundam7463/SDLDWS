Documentação SDLDWS.

Sobre

Essa documentação tem como objetivo auxiliar os usuários do software SDLDWS. A SDLDWS tem dois tipos de usuário, o desenvolvedor quem vai trabalhar mais afundo no framework, alterando o código fonte, adicionando novas funcionalidades e o usuário final, o que vai criar os jogos e testar.
